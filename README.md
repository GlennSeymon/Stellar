## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

Project Motivation - I had multiple Stellar wallet each containing hundreds of assets that I wanted to swap to XLM and remove the trustlines. This would have taken weeks using the Lobstr application.

Using the <a href="https://github.com/elucidsoft/dotnet-stellar-sdk">dotnet-stallar-sdk</a>, I was able to write this console application to perform these operations using the secretKey provided.
